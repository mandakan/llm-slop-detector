LLM Slop Detector

Flags invisible Unicode, AI-style punctuation, and telltale LLM phrases in any textarea on the web.

Read-only page scanning

When on, the toolbar icon shows a "Scan this page" button that highlights slop in any webpage you're reading (not just textareas). Off by default if you only want to lint your own writing.

Rule packs

Core rules are always on. Packs add extra model-specific or domain-specific phrases.

Disabled sites

Per-site toggles live in the extension's action popup. Hosts you've turned off:

    Privacy

    • No network requests. Ever. All scanning runs locally.
    • No telemetry. No analytics.
    • Settings are stored in chrome.storage.local. Not synced.
    • Nothing you type is logged, persisted, or transmitted.
