# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The short version: be respectful, assume good intent, keep discussions about the work. If you would not say it in a professional setting, do not say it here.

## Reporting

If you experience or witness behavior that violates the Contributor Covenant in this project's issues, PRs, or discussions, please report it privately to the maintainer via [GitHub's private vulnerability reporting](https://github.com/mandakan/llm-slop-detector/security/advisories/new) (it doubles as a general private channel) or by email at the address listed in the git history.

Reports will be handled confidentially. The maintainer may remove content, lock threads, or block accounts as needed.
